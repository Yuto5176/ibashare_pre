# 高齢者のための居場所発見・共有支援アプリ「#いばしぇあ」

「#いばしぇあ」は、自分らしくいられる場所や活動の発信・共有を容易にし、函館市の高齢者の活動を盛り上げる手助けを行うAndroidアプリです。

高齢者の方のフィードバックを反映しながら開発を続けています。
